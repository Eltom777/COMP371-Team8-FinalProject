-------------------------
	KEY BINDS
-------------------------

* CHOOSE A MODEL (1-5) TO APPLY TRANSFORMATIONS / RENDERING MODE:

	Top left:     1
	Top right:    2
	Bottom left:  3
	Bottom right: 4
	Middle:       5

* MODEL TRANSFORMATIONS:
	
	Shift left:   A
	Shift right:  D
	Shift up:     W
	Shift down:   S

	Scale up:     U
	Scale down:   J

	Rotate right: Minus (-)
	Rotate left:  Equal (=)

* RENDERING MODE:

	Points:       P
	Lines:        L
	Triangles:    T
	Textures:     X

* CAMERA:
	
	Rx:		  Left arrow
	R-x:	 	  Right arrow
	Ry:		  Up arrow
	R-y:	 	  Down arrow
	
	Pan:		  Right mouse button + movement in x direction
	Tilt:		  Middle mouse button + movement in y direction
	
	Zoom in:	  Left mouse button + mouse movement down OR mouse scroll up
	Zoom out:	  Left mouse button + mouse movement up OR mouse scroll down
